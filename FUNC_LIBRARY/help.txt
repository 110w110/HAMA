1. 조회시스템
<조회시스템이란>
원하는 기간과 키워드에 알맞게 조회하는 시스템입니다.

<조회시스템 사용방법>
회계 구분과 결의서를 선택하고 원하는 기간과 검색키워드를 입력하시면 해당하는 자료를 전체 조회합니다.
</LOOKUP>

<WRITE>
2. 작성시스템
<작성시스템이란>
'data.xlsx'에 작성된 내용을 바탕으로 회계정보를 시스템에 자동으로 작성합니다.
작성시스템 실행 시, 사이트에 접속하는 동안 첨부파일 등록을 위한 폴더 정리가 시작됩니다.

<작성시스템 사용방법>
1) 자동으로 작성하기 위한 모든 내용을 'data.xlsx' 파일에 작성합니다.

<엑셀파일>
'data.xlsx' 파일 사용과 관련된 내용은 파일 내부 메모 기능을 통해 설명되어 있습니다.
해당 파일의 대부분의 셀은 모두 수식으로 작성되어, 한번 작성한 후 지우게 되면 기능을 잃습니다.
따라서 사용하신 셀 또는 행을 지우실 경우, 행 삭제를 통해 한 줄씩 모두 지우시는 것을 권장합니다.
기존에 준비된 서식은 약 500개 가량의 행이 준비되어 있으므로, 모든 행을 다 사용하신 경우에는 백업파일을 복구하여 사용하시기 바랍니다.

<첨부파일 등록 과정>
첨부파일 양식 : '(엑셀 내 키워드를 포함한 제목)'
첨부파일 예시 : '스튜디오 관리비 6월.pdf'
            '스튜디오 6월'
                '스튜디오 운영비 6월'
키워드 예시 : '스튜디오 관리비' -> out에 스튜디오 폴더 생성
첨부파일 등록을 위의 고유한 키워드에 알맞게 이름을 작성하시어 'in' 폴더에 포함시키면 작성시스템 실행과 동시에 자동으로 'out' 폴더의 적절한 폴더로 재배치됩니다.
</WRITE>

<MODIFY>
3. 수정시스템
<수정시스템이란>
기존에 작성된 내용을 바탕으로 날짜만 수정하여 재작성하기 위한 시스템입니다.
원하는 결의서를 클릭한 뒤 프로그램에서 엔터키를 누르면 복사하여 다음 달로 작성합니다.

<수정시스템 사용방법>
수정시스템 실행 시, 회계 정보를 조회하는 창이 나타납니다.
재작성을 원하는 자료를 열람한 후 엔터를 입력하시면 기존 정보의 날짜 정보의 규칙성을 파악해 적절한 날짜로 수정을 진행합니다.
여러 가지 예외의 경우가 있어 원하는 날짜로 수정되지 않은 경우, 원하는 날짜로 직접 수정하신 후 다시 엔터를 입력하시면 저장을 진행합니다.

<수정시스템 날짜수정규칙>
1) 기본적으로 n월은 n+1월로 변경합니다
ex) 4월분 -> 5월분
ex) 수도세 3월, 전기세 4월 -> 수도세 4월, 전기세 5월

2) 연속된 월은 시작한 월에서 끝나는 월의 차이를 파악하여 변경합니다.
ex) 4월,5월 -> 6월,7월
ex) 3월,4월,7월 -> 8월,9월,11월
ex) 3,4월 -> (현재 버전에서는 지원하지 않습니다)

3) 물결표를 통한 연속된 월 또한 시작한 월과 끝나는 월의 차이를 파악하여 변경합니다.
ex) 3월~6월 -> 7월~9월
ex) 10월~12월 -> 2023년1월~2023년3월

4) 변경 후의 월이 한 해를 넘어간 경우 기본적으로 오늘의 연도로 생각하고 다음 해로 변경합니다.
만약 연도가 입력되어 있다면 그 연도를 기준으로 다음 해로 변경합니다.
ex) 12월 -> 2023년 1월
ex) 2019년 12월 -> 2020년 1월

5) 구체적인 날짜로 작성되어 있으면 해당 날짜 간 차이를 단순 계산하여 변경합니다.
ex) 2021.6.17~2022.6.16 -> 2022.6.17~2023.6.16 (1년으로 계산)
ex) 2021.6.17~2021.9.16 -> 2021.9.17~2021.12.16 (3개월로 계산)

이외에 기본적인 날짜 계산은 구현되어 있으나 날짜와 날짜 사이에 공백이 존재하면 다른 날짜로 인식하게 되는 경우가 있습니다.
</MODIFY>
